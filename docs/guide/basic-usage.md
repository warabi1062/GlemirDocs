## カメラ操作

アバタービュー上でマウスを使ってカメラの位置やズームを操作できます。

| 操作 | 動作 |
|------|------|
| ドラッグ | カメラの XY 移動 |
| スクロール | ズームイン / ズームアウト |

設定パネルの Camera セクションからも X / Y / Z の数値入力と FOV スライダーで調整可能です。詳細は [設定パラメータ](./settings) を参照してください。

## キャリブレーション

カメラに正面を向いた状態で、以下のいずれかの方法でキャリブレーションを実行します。

- 設定パネルの「Calibrate」ボタンをクリック
- `R` キーを押下

キャリブレーションにより、現在の頭部の位置と向きが「正面」として設定されます。姿勢を変えたときやカメラの位置を調整した後は、再度キャリブレーションを行ってください。

## フェイストラッキング

Glemir は MediaPipe を使用して以下の動きを検出し、アバターに反映します。

### 頭部の動き

カメラに映った顔の向き（上下・左右・傾き）をアバターの Head / Neck ボーンに反映します。One Euro Filter によるスムージングが適用されており、自然な動きになります。

### 上半身の動き

頭部の回転に連動して、Spine / Chest ボーンが緩やかに追従します。追従の度合いは設定パネルの「Body Follow Ratio」で調整できます。

### 表情

以下の表情がトラッキングされます。

| 表情 | 説明 |
|------|------|
| まばたき | 左右独立で検出 |
| リップシンク | 5母音（あ・い・う・え・お）に対応 |
| Happy | 笑顔の表情 |
| Angry | 怒りの表情 |
| Sad | 悲しみの表情 |
| Surprised | 驚きの表情 |

::: info
感情表現（Happy / Angry / Sad / Surprised）は排他制御されており、同時に複数が有効になることはありません。感度は「Emotion Sensitivity」パラメータで調整できます。
:::

## キーボードショートカット

| キー | 操作 |
|------|------|
| `R` | キャリブレーション実行 |
| `F1` | デバッグオーバーレイ（BlendShape 値の表示）の切替 |

## デバッグ機能

### BlendShape デバッグオーバーレイ

`F1` キーを押すと、画面上に各 BlendShape の値をバーグラフで表示するオーバーレイが出現します。トラッキングの状態を確認したい場合に使用してください。

### Web カメラプレビュー

メニューバーの Debug メニューからカメラプレビューの表示を切り替えられます。有効にすると、画面右上に Web カメラの映像と 468 点の顔ランドマークがオーバーレイ表示されます。
