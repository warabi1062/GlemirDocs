# 設定パラメータ

設定パネルは **Camera**・**Avatar**・**Lighting** の3つのセクションに分かれています。各スライダーはダブルクリックでデフォルト値にリセットできます。

## Camera

カメラの画角と位置を調整します。

### FOV

| 項目 | 値 |
|------|------|
| 範囲 | 15 〜 80 |
| デフォルト | 60 |

カメラの視野角（Field of View）。値を小さくするとアバターが大きく表示され、大きくすると広角になります。

### カメラ位置（X / Y / Z）

アバターに対するカメラの位置を数値入力で調整します。

::: tip
アバタービュー上でドラッグ操作でも移動できます。
:::

## Avatar

トラッキングの反応や動きの強さを調整します。

### Body Follow Ratio

| 項目 | 値 |
|------|------|
| 範囲 | 0 〜 1 |
| デフォルト | 0.3 |

頭部の回転に対する上半身の追従比率。0 にすると上半身は動かず、1 にすると頭部と同じ量だけ回転します。

### Head Rotation Scale

| 項目 | 値 |
|------|------|
| 範囲 | 0 〜 1 |
| デフォルト | 1.0 |

頭部回転の倍率。値を大きくするほど頭の動きが大きくなり、小さくすると控えめになります。

### Head Roll Reduction

| 項目 | 値 |
|------|------|
| 範囲 | 0 〜 1 |
| デフォルト | 0.6 |

頭部のロール（左右の傾き）の軽減率。0 だとロールがそのまま反映され、1 にするとロールが完全に無効化されます。

### Emotion Sensitivity

| 項目 | 値 |
|------|------|
| 範囲 | 0 〜 1 |
| デフォルト | 0.5 |

感情表現（Happy / Angry / Sad / Surprised）の検出感度。値を大きくするほど感情が反映されやすくなり、小さくすると反映されにくくなります。0 にすると感情表現は無効になります。

## Lighting

シーンの照明を調整します。

### Color Temperature

| 項目 | 値 |
|------|------|
| 範囲 | 2,000 〜 10,000 |
| デフォルト | 6,500 |

ライトの色温度（ケルビン）。低い値で暖色（電球色）、高い値で寒色（昼光色）になります。

### Light Intensity

| 項目 | 値 |
|------|------|
| 範囲 | 0 〜 2 |
| デフォルト | 1.0 |

ライトの明るさ。値を大きくするとアバターが明るくなります。
